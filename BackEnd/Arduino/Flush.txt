Flush Arduino Board:
--------------------
- press <Windows>+R and put in "cmd" and press <OK>
- go to directory "avrdude"
- start listComPorts.exe
- in the ouput there is the com port where the arduino is. 
- set the com port in the file installFW.bat/installFW.sh in line SET COMPORT=<com port>
- call skript installFW.bat/installFW.sh
